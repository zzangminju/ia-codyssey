## 1. 브랜드 아이덴티티

### 핵심 키워드: **"우리(We)"**

이 대본 전체의 구조적 장치는 **브랜드명 자체가 철학**이라는 점을 활용합니다.

> "우리은행"에서 **'우리'**를 제거하면 차갑고 기능적인 '은행'만 남는다.

'우리(We/Us)'는 단순한 고유명사가 아닌, **공동체·관계·연대**를 뜻하는 언어적 장치로 작동합니다. 이름 안에 이미 브랜드 가치가 내재되어 있습니다.

---

### 브랜드 포지셔닝

| 대비 구도 | 타 금융 (암묵적) | 우리은행 |
|-----------|----------------|---------|
| 관계 | 거래(Transaction) | 관계(Relationship) |
| 대상 | 숫자·실적 | 사람 |
| 감성 | 차갑고 기계적 | 따뜻하고 인간적 |
| 역할 | 금융 서비스 제공자 | 삶의 동반자 |

→ **"사람 중심 금융"** 포지셔닝을 경쟁 차별점으로 설정

---

### 메인 타겟

**20~40대, 금융을 "필요하지만 차갑고 불편한 것"으로 느끼는 사람들**

단순한 연령 구분이 아닌 **심리적 타겟**으로 정의하는 게 핵심입니다.

| 구분 | 내용 |
|------|------|
| 1차 타겟 | 사회 초년생 / 청년 (22~35세) |
| 2차 타겟 | 소상공인 / 자영업자 (35~50세) |
| 심리적 공통점 | "은행은 나를 숫자로 본다"는 피로감을 가진 사람 |
| 행동적 특징 | 브랜드의 가치관에 공감할 때 선택하는 세대 |

---

### TONE & MANNER — 어떻게 말하는가

```
차갑지 않게  ────────────────────  과하지 않게
    냉소 ×                            눈물 자극 ×
                     ↓
           우리은행이 있어야 할 자리
            조용하지만 단단한 온기
```

| 톤 요소 | 방향 |
|---------|------|
| 언어 톤 | 설득하지 않고 **공감**한다 |
| 감정 온도 | 뜨겁지 않고 **따뜻하게** |
| 시각 톤 | 화려하지 않고 **정갈하게** |
| 음악 톤 | 웅장하지 않고 **서정적으로** |
| 자막 폰트 | 고딕 계열, **신뢰감 + 단정함** |

❌ 피해야 할 톤: 눈물 유발형 감동 광고 / 스펙·금리 나열형 / 유머·발랄 계열

---

### USP — 왜 우리은행인가

> **"우리은행은 이름 자체가 철학이다. '우리(We)'라는 단어를 브랜드로 가진 유일한 은행."**

| 비교 | 타 은행 | 우리은행 |
|------|---------|---------|
| 브랜드 차별화 방식 | 상품/금리/혜택 강조 | **이름의 의미**로 차별화 |
| 감성 접근 | 행복한 가족 이미지 | **"없다면 어떻게 될까"** 결핍 역설 |
| 공동체 표현 | 카피로만 언급 | 브랜드 네임 자체가 공동체 |

---

### 최종 브랜드 아이덴티티

> **"우리은행은 '우리(We)'라는 이름처럼, 숫자가 아닌 사람을 중심에 두고 공동체와 함께 존재하는 금융 파트너다."**

**광고 목적/핵심 메시지:** '우리'라는 이름의 의미를 재발견하게 함으로써, 우리은행을 단순한 금융기관이 아닌 공동체의 동반자로 인식시키는 브랜드 감성 광고.

| 요소 | 내용 |
|------|------|
| 광고 목적 | 브랜드 인지 + 감성적 이미지 재정의 |
| 핵심 타겟 인사이트 | "우리은행"을 그냥 은행으로만 알고 있는 사람들 |
| 핵심 메시지 1개 | 우리은행 = 공동체의 동반자 |
| 설득 방식 | 논리가 아닌 **결핍의 감정** ("없다면 어떻게 될까") |

## 2. 광고 대본 (수정안)

> 총 30초 · 5개 씬

### 씬 1 (10초)

**화면**
AI로 생성한 거대하고 웅장한 우리은행 본점 건물 외관. 정문에 붙은 "우리은행 로고 + 우리은행" 간판에서 "우리은행 로고 + 우리" 부분의 빛이 꺼진다. 그냥 "은행"으로 표시됨. 불이 꺼지면서 주변이 어두워지거나 색감의 채도가 빠지며 차갑게 변하는 효과.

**자막 / 나레이션**
> "만약 은행에서 '우리'라는 단어가 사라진다면 어떻게 될까요?"

**오디오 / 음악:** 건조하고 낮은 기계음

---

### 씬 2 (6초)

**화면**
화면이 전환되며 소상공인, 취약계층, 청년들이 서로를 돕고 미소 짓는 따뜻한 일상이 빠르게 지나감 (AI 생성 스틸컷).

**나레이션**
> "상인의 땀방울을 닦아주고, 청년의 내일을 안아주며, 이웃의 어려움을 함께 나누는 '우리'가 없다면,"

---

### 씬 2.5 (4초)

**화면:** 차가운 느낌의 은행 내부

**나레이션**
> "은행은 그저 차가운 건물이 될 뿐입니다."

---

### 씬 3 (5초)

**화면**
지워졌던 간판에 '우리은행 로고 + 우리'라는 글자가 다시 켜진다. 글자가 따뜻한 황금빛, 부드러운 화이트 톤으로 서서히 차오른다. '우리은행 로고 + 우리은행' 완성. 처음에 로고가 없던 자리는 비워둠.

**나레이션**
> "숫자가 아닌 사람을 담는 곳. 거래가 아닌 관계를 쌓는 곳. 우리은행은 언제나 우리의 곁에 있겠습니다."

**오디오 / 음악:** 음악 고조, 따뜻한 현악기 선율

---

### 씬 4 (5초)

**배경:** 따뜻한 화이트/크림 톤

**화면:** 우리은행 로고 + "우리 마음속 첫번째 금융"

**자막:** "우리은행, 우리 마음속 첫번째 금융"

---

### 전체 공통 자막 가이드

- **폰트:** 신뢰감을 주는 고딕 계열의 정갈한 흰색 폰트
- **위치:** 화면 약간 아래쪽 고정
- **색감 강조:** 씬 3의 '우리은행' 복구 장면은 따뜻한 주황/금빛(Warm Tone)으로 확실하게 대비

## 3. 스토리보드 — AI 도구 활용 계획 A (Midjourney)

> 이미지: **Midjourney v6** · 영상: **Google Veo 2** · 나레이션: **ElevenLabs** · 음악: **Suno v4**

### 씬 1 — 우리은행 본점 외관 간판 소등 (10초)

| 항목 | 내용 |
|------|------|
| 화면 연출 | '우리은행' 간판에서 '우리' 글자의 빛이 서서히 꺼짐 → 주변 채도 저하, 차가운 청회색 색조로 전환 |

**도구 선택 이유**

- **Midjourney v6** — 고층 건축물의 야경·유리 반사 포토리얼 렌더링에 특화. `--ar 16:9 --style raw`로 시네마틱 스틸컷 확보
- **Veo 2** — 스틸→모션 변환 시 간판 조명 소등 애니메이션과 카메라 느린 줌아웃을 프롬프트로 직접 제어 가능
- **Suno v4** — `"low drone, industrial, cold"` 프롬프트로 건조한 기계음 생성 → 감성 대비 포인트 확보

---

### 씬 2 — 소상공인·청년·취약계층 따뜻한 일상 (6초)

| 항목 | 내용 |
|------|------|
| 화면 연출 | 소상공인 · 청년 · 취약계층 스틸컷 3장이 빠르게 컷 전환. 웜톤 자연광. 서로를 돕고 미소 짓는 장면 |

**도구 선택 이유**

- **Midjourney v6** — 인물 감성 장면에서 `"warm candid, golden hour"` 스타일로 광고급 스틸컷 생성. 다양한 연령·직업 표현에 강함
- **Veo 2** — Image-to-Video 기능으로 스틸컷에 자연스러운 숨결·미세 움직임 부여. 컷 리듬을 프롬프트로 지정
- **ElevenLabs** — 한국어 감성 나레이션에 최적화. 목소리 톤(따뜻함·진정성)을 슬라이더로 세밀하게 조절 가능

---

### 씬 2.5 — 차가운 은행 내부 (4초)

| 항목 | 내용 |
|------|------|
| 화면 연출 | 채도 제거된 냉랭한 은행 내부. 텅 빈 창구, 형광등, 무표정한 공간감. 씬2의 따뜻함과 극대 대비 |

**도구 선택 이유**

- **Midjourney v6** — `"desaturated, cold fluorescent, empty bank interior"` 프롬프트로 씬2와 명확한 색온도 대비 구현
- **Veo 2** — 느린 달리(dolly) 카메라 이동으로 텅 빈 공간의 공허함 강조. 4초 내 감성 전달에 적합

---

### 씬 3 — 간판 복구 황금빛 점등 (5초)

| 항목 | 내용 |
|------|------|
| 화면 연출 | '우리' 글자가 따뜻한 황금빛으로 서서히 채워짐 → 할로(halo) 효과 → '우리은행' 완성. 주변 색온도 웜톤 전환 |

**도구 선택 이유**

- **Midjourney v6** — 씬1과 동일한 구도·시드값 재사용 → 간판 소등·점등 전후 장면의 시각적 일관성 보장
- **Veo 2** — 조명 fill-in 애니메이션과 색온도 변화를 단일 샷으로 처리. `"warm light fills signage slowly"` 프롬프트 지원
- **Suno v4** — `"strings swell, warm orchestral, crescendo"` 프롬프트로 씬1의 기계음과 대비되는 현악기 고조 구현

---

### 씬 4 — 로고 + 슬로건 엔딩 (5초)

| 항목 | 내용 |
|------|------|
| 화면 연출 | 따뜻한 화이트/크림 배경. 우리은행 로고 부드럽게 페이드인. 슬로건 자막 등장. 음악 자연스럽게 페이드아웃 |

**도구 선택 이유**

- **Veo 2** — 로고 등장 모션(페이드인·스케일업)과 배경 색상 전환을 단일 클립으로 출력. 별도 합성 후처리 불필요
- **Suno v4** — 씬3에서 이어지는 현악 선율의 자연스러운 페이드아웃·엔딩 처리를 동일 트랙 내 연장으로 해결

## 4. 스토리보드 — AI 도구 활용 계획 B (나노바나나)

> 이미지: **나노바나나** · 영상: **Google Veo** · 나레이션: **ElevenLabs** · 음악: **Suno**

*한국어 프롬프트 최적화가 필요한 경우 Midjourney 대신 국산 이미지 AI인 나노바나나를 활용하는 대안 플랜입니다.*

### 씬 1 — 우리은행 본점 간판 소등 (10초)

**도구 선택 이유**

- **나노바나나** — 한국 건물 특유의 질감과 간판 레이아웃을 한국어 프롬프트로 정밀 생성. 레퍼런스 스틸컷 제작
- **Veo** — 스틸컷 기반 이미지-투-비디오로 간판 소등 → 채도 저하 애니메이션을 자연스러운 카메라 워크와 함께 생성
- **ElevenLabs / Suno** — ElevenLabs: 신뢰감 있는 한국어 나레이션 보이스 / Suno: 건조하고 낮은 기계음 BGM 생성

---

### 씬 2 — 따뜻한 일상 소상공인·청년·취약계층 (6초)

**도구 선택 이유**

- **나노바나나** — "시장 상인", "취업 준비 청년" 등 한국 생활 맥락에 밀착된 인물 스틸컷을 높은 사실감으로 생성. 3~4장 세트 제작
- **Veo** — 스틸컷에 자연스러운 줌·패닝 모션을 부여하고, 빠른 컷 편집에 적합한 짧은 클립(1~2초) 단위로 분할 생성

---

### 씬 2.5 — 차가운 은행 내부 (4초)

**도구 선택 이유**

- **나노바나나** — 씬1과 동일한 시각적 언어(채도 저하, 블루 톤)를 유지하는 실내 스틸컷 생성. 프롬프트 일관성 확보
- **Veo** — 느린 달리 숏으로 공간의 냉랭함을 극대화. 단 4초 분량의 짧은 클립도 프레임-레벨 제어로 정밀 생성 가능

---

### 씬 3 — "우리" 복구 황금빛 점등 (5초)

**도구 선택 이유**

- **나노바나나** — 씬1에서 생성된 건물 스틸컷을 재활용. "점등 후" 상태(황금빛 조명 버전)를 별도 프롬프트로 생성해 Before/After 쌍 확보
- **Veo** — 두 스틸컷(소등→점등) 간 모핑 트랜지션을 비디오로 생성. 빛이 차오르는 연속적 애니메이션은 비디오 AI가 필수
- **ElevenLabs / Suno** — 감정이 고조되는 클라이맥스 나레이션 톤 조절 / 현악기 고조 BGM을 씬1 기계음과 동일 세계관으로 이어지도록 생성

---

### 씬 4 — 엔딩 브랜드 슬로건 (5초)

**도구 선택 이유**

- **나노바나나** — 브랜드 가이드라인에 맞는 크림 배경 + 슬로건 레이아웃 스틸컷 제작. 후보 컷 여러 장 빠르게 생성 후 선별
- **Veo** — 스틸컷에 로고 페이드인 + 슬로건 타이핑 모션 효과 부여. 정적인 엔딩에 안정적인 마무리 움직임 추가
- **Suno** — 현악기 선율을 자연스럽게 감소시키며 마무리되는 아웃트로 구간 생성. 전체 BGM의 시작~끝을 단일 프로젝트로 통일
